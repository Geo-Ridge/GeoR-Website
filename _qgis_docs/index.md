## Available Plugins

GeoRidge provides several QGIS plugins for enhanced geospatial analysis:

- **Data Importer**: Import geospatial data from multiple sources
- **Map Composer**: Advanced map composition and printing
- **Analysis Tools**: Specialized geospatial analysis tools
